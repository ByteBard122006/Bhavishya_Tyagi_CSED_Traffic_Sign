## 📄 Description
This mini project uses Python and machine learning techniques to detect traffic signs from images.  
It aims to help autonomous systems understand road signs for safer and smarter navigation.

## 🛠️ Technologies Used
- Python 🐍
- OpenCV
- NumPy
- Machine Learning (ML)
